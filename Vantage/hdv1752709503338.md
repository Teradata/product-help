The Nvidia NIM models are accessed through a **ModelOps** project type that supports Nvidia models. You cannot use the NIM project with your BYOM or Python models.

You can track deployments, manage endpoints, and monitor performance through the [ModelOps Dashboard](bvr1694122081407.md).

1.  From the **AI Workbench** home page, select **Open ModelOps**.


1.  Enter your user credentials.


1.  Select **Projects**.


1.  Select a NIM project or enter NIM in the **Search** bar to find a NIM-specific project.


1.  Select a Nvidia NIM model from the **Models** tab.

    **Hint:** To go back to the model list and select a different model, select **NIM** from the header path.
