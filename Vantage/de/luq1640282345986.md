**Hinweis:** Um Daten in eine andere als Ihre eigene Datenbank zu laden, benötigen Sie diese Berechtigungen für die andere Datenbank.

-   SELECT

-   UPDATE

-   LÖSCHEN

-   INSERT

-   CREATE TABLE

-   EXECUTE WITH GRANT OPTION
