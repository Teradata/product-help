1.  Wählen Sie im ModelOps-Dashboard das Projekt > **Merkmalsengineering** > **Aufgabe erstellen** aus.


1.  Füllen Sie die Felder der Registerkarte **Basis** aus:

    -   Geben Sie der Aufgabe einen aussagekräftigen **Namen** und eine **Beschreibung**.


    -   Geben Sie die **Sprache** der zum Erstellen dieser Aufgabe verwendeten Skriptdatei an. Derzeit wird nur Python unterstützt.


    -   Geben Sie die **Skriptquelle** an – Laden Sie die Datei hoch, indem Sie nach ihr suchen, sie per Drag & Drop einfügen oder den Git-Speicherort (das im Projekt definierte Git-Repository) angeben.


    -   Geben Sie den aufzurufenden **Funktionsnamen** an, z. B. „run_task“.


1.  Füllen Sie die Felder auf der Registerkarte **Erweitert** aus:

    -   Wählen Sie das **Docker-Image** aus.


    -   Wählen Sie die **Ressourcenvorlage** (Dataset-Vorlage) aus.

        **Hinweis**: Wenn Ihre Umgebung Knoten-Pools verwendet, siehe [Knoten-Pool auswählen](jbj1778560430063.md).


    -   Legen Sie beliebige **Schlüssel**/**Wert**-Paar-Informationen unter **Benutzerdefinierte Eigenschaften** fest.


1.  Wählen Sie **Erstellen** aus.
