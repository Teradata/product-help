Verfügbar auf AWS.

Thales CipherTrust Database Protection (CDP) auf VantageCloud Lake bietet benutzerdefinierte Funktionen (User Defined Functions, UDFs), die eine leistungsstarke Verschlüsselung auf Spaltenebene für strukturierte Daten in Datenbanken ermöglichen. Durch die effiziente Ver- und Entschlüsselung bestimmter Spalten in Datenbanken können große Mengen sensibler Daten in die Datenspeicher verschoben und aus diesen entfernt werden. CDP kann skaliert werden, um mehrere Rechenzentren in lokalen, virtuellen und öffentlichen Cloud-Umgebungen zu unterstützen.

Für die Aktivierung von Thales in Ihrer Umgebung benötigen Sie Folgendes:

-   Eine gültige Lizenz für Ihren CipherTrust Manager-Server.


-   Auf Ihren CipherTrust Manager-Server kann über einen VPC-Service-Endpunkt (Virtual Private Cloud) unter Verwendung von PrivateLink zugegriffen werden.


-   Die **Verfügbarkeitszonen-ID** der VantageCloud Lake-Umgebung und die AWS-**IAM-Rolle**.


Teradata empfiehlt die Verwendung eines einzelnen CipherTrust Manager-Servers mit einer einzelnen VantageCloud Lake-Umgebung.

Wählen Sie im Menü **Startseite** > **Benutzerdefinierte Funktionen** > auf der Registerkarte **Drittanbieteranwendungen** > **Thales** unter dem Abschnitt **Verfügbare Partnerlösungen** aus.

Füllen Sie das Formular aus und wählen Sie **Installieren**aus.

Sie können jederzeit die Installation abbrechen, die Konfiguration bearbeiten oder das Thales-Paket deinstallieren.
