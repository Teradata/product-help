When you run a job in ModelOps, you select a node pool and configure the resources for that job. Selecting the right compute resources in ModelOps depends on the type of workload, data size, and performance requirements.

|Option|Description|Recommended use cases|
|-------|------------|----------------------|
|General compute|5 nodes of 16 vCPU, memory: 64 GB RAM|Best for exploration, analysis, in-database functions|
|High memory|3 nodes of 64 vCPU, memory: 512 GB RAM|Best for large datasets, memory-intensive workloads|
|GPU accelerated|1 node of 8xNVIDIA H200, 64 CPU, memory: 503 GB RAM|NVIDIA H200 optimized for inference services|
