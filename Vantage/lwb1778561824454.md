After you select a node pool, configure the resources for the job. Two options are available:

-   **Standard**

    Applies default resource values suitable for basic workloads. For example, 1 vCPU and 2 GB memory.


-   **Custom**

    Adjust the resource allocation using the slider, up to the maximum allocatable value for the selected node pool. For example, up to 8 vCPU and up to 32 GB memory.
