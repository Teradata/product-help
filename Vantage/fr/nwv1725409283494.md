Les modèles permettent d'organiser les ensembles de données et de faciliter leur création en héritant de tous les paramètres définis. Les modèles d'ensembles de données peuvent également être utilisés pour créer et regrouper des ensembles de données liés à un cas d'utilisation.

Pour créer un ensemble de données à partir d'un modèle, utilisez les paramètres par défaut définis pour le modèle ou mettez-les à jour au besoin.
