Vous pouvez calculer des statistiques pour vos caractéristiques et vos cibles lorsque vous créez ou modifiez un modèle d'ensemble de données. Le stockage des informations statistiques relatives aux caractéristiques est nécessaire pour assurer le suivi de la dérive des données.

1.  Reportez-vous à [Créer un modèle d'ensemble de données](frd1725409311264.md) pour connaître les étapes d'accès à l'onglet **Caractéristiques**.


1.  Sous l'onglet **Caractéristiques**, après avoir exécuté la requête SQL pour récupérer les caractéristiques, la table des caractéristiques affiche toutes celles qui sont disponibles. Sélectionnez **Valider les métadonnées**.

    ModelOps affiche un avertissement pour les statistiques manquantes des cibles et la possibilité de générer ou de regénérer des statistiques pour la cible sélectionnée.


1.  Pour générer des métadonnées, sélectionnez le type de caractéristique pour chaque caractéristique à inclure en cochant la case correspondante dans la table.


1.  Sélectionnez **Générer des métadonnées** pour générer des métadonnées pour les caractéristiques qui n'en contiennent pas.

    Sélectionnez **Regénérer les métadonnées** pour mettre à jour les métadonnées existantes.


1.  Vérifiez les noms des caractéristiques dans la boîte de dialogue, puis sélectionnez **Générer les métadonnées** ou **Regénérer les métadonnées**.

    La tâche de calcul des statistiques a démarré.


1.  Sélectionnez ![icône Fermer](Images/teg1680569591203.svg) une fois la progression du calcul des statistiques terminée.


1.  Une fois les métadonnées valides disponibles, sélectionnez **Afficher les détails** pour afficher les statistiques calculées.

    Le même processus peut être également appliqué à la cible.


Pour BYOM, une tâche de calcul des statistiques s'exécute après l'importation du modèle. Pour l'exécuter manuellement pour un modèle BYOM, cliquez avec le bouton droit sur une version du modèle. > **Calculer les statistiques**.
