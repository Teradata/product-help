Vous devez attribuer un [modèle d'ensemble de données](frd1725409311264.md) et un [ensemble de données](xfu1732652871944.md).

1.  Dans le tableau de bord ModelOps, sélectionnez le projet > **Modèles** > la version du modèle > **Évaluer**.

    Sélectionnez une connexion personnelle d'ensemble de données dans la liste déroulante. Si aucune connexion personnelle d'ensemble de données par défaut n'a été créée, vous serez invité à le faire.

    1.  Renseignez les champs et enregistrez la connexion d'ensemble de données.

        Une fois créée, la connexion d'ensemble de données s'affiche dans le menu.


    1.  Sélectionnez **Continuer**.


1.  Définissez les propriétés dans les onglets **De base** et **Avancé**.

    1.  Dans l'onglet **De base**, sélectionnez le **Modèle d'ensemble de données** et l'**Ensemble de données**, et définissez manuellement les variables d'entraînement dans **Hyperparamètres** avec une valeur prédéterminée avant de démarrer la tâche d'entraînement.


    1.  Dans l'onglet **Avancé**, spécifiez la version de l'**Image de Docker** et les détails du **Modèle de ressource**. Si votre environnement utilise des pools de nœuds, reportez-vous à [Sélectionner un pool de nœuds](jbj1778560430063.md).

        Identifiez un ensemble prédéfini de ressources, y compris le CPU et la mémoire, qui sont les propriétés du conteneur créé dans lequel exécuter la tâche.

        -   Sélectionnez une taille de **Modèle** dans la liste déroulante.


        -   Définissez l'une des propriétés suivantes :

            **Mémoire** : spécifie la mémoire RAM allouée au conteneur. Cette valeur doit être un nombre entier suivi de l'unité (m pour mégaoctets ou g pour gigaoctets).

            **CPU **: spécifie les unités CPU (cœurs) allouées au conteneur. Il peut s'agir d'un nombre entier ou décimal. Accepte m comme unité pour les millicœurs.

            **GPU** (visible lorsque l'option **Personnalisé** est sélectionnée) : spécifie les unités GPU allouées au conteneur, peut être un nombre entier ou décimal. Accepte m comme unité pour les millicœurs.


1.  Sélectionnez **Évaluer le modèle**.

    L'option **Évaluer les détails de la tâche** affiche les informations sur les journaux. Vous pouvez **arrêter** (suspendre) la progression.

    L'option **Journaux** répertorie les journaux et les événements pour la tâche sélectionnée.

    La section **Événements** dans l'onglet **Journaux** affiche la progression des événements.

    L'option **Propriétés** répertorie toutes les propriétés liées à la tâche sélectionnée, telles que l'ID de tâche, l'ID d'ensemble de données et les hyperparamètres.


1.  Sélectionnez ![icône Fermer](Images/teg1680569591203.svg) une fois la progression de l'évaluation terminée.

    La liste des versions de modèle présente les détails de chaque version du modèle.
