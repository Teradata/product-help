1.  Dans le tableau de bord ModelOps, sélectionnez le projet > **Ingénierie des caractéristiques** > **Créer une tâche**.


1.  Renseignez les champs de l'onglet **De base** :

    -   Donnez à la tâche un **Nom** et une **Description** significatifs.


    -   Spécifiez le **Langage** du fichier de script utilisé pour créer la tâche. Actuellement, seul Python est pris en charge.


    -   Spécifiez la **Source du script** : chargez le fichier en y accédant, en le glissant-déplaçant ou en spécifiant l'emplacement de Git (référentiel Git défini dans le projet).


    -   Spécifiez le **Nom de la fonction** à appeler tel que run_task.


1.  Renseignez les champs de l'onglet **Avancé** :

    -   Sélectionnez l'**Image de Docker**.


    -   Sélectionnez le **Modèle de ressource** (modèle d'ensemble de données).

        **Remarque** : si votre environnement utilise des pools de nœuds, reportez-vous à [Sélectionner un pool de nœuds](jbj1778560430063.md).


    -   Définissez les informations sur la paire **Clé**/**Valeur** dans **Propriétés personnalisées**.


1.  Sélectionnez **Créer**.
