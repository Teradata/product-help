Vous pouvez afficher l'historique des validations pour les modèles basés sur Git du projet sélectionné. L'historique des validations fournit une chronologie des modifications apportées au code du modèle.

Dans le tableau de bord ModelOps, sélectionnez le projet > **Modèles** > le modèle Git > développez une ligne pour examiner les détails de la version du modèle affectés par une validation.
