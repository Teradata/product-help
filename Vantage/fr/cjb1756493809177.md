Un administrateur d'organisation peut gérer l'autorisation des périphériques OAuth 2.0. Cette option contrôle l'accès à l'API REST d'Open Analytics Framework de la console pour tous les environnements provisionnés dans l'organisation. Lorsqu'elle est désactivée, vous pouvez utiliser un jeton d'accès personnel (PAT) pour accéder aux API REST d'Open Analytics.

Sur la page **Accueil**, sélectionnez le menu ![menu kebab](Images/zsz1597101912145.svg) d'OAuth pour désactiver ou activer OAuth 2.0.
