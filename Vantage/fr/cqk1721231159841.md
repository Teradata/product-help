Vous pouvez configurer votre environnement VantageCloud Lake pour que d'autres utilisateurs et applications puissent y accéder par le biais d'une connexion **Internet** ou d'une connexion **Liaison privée**.

**Remarque :** Les utilisateurs peuvent toujours se connecter directement à VantageCloud Lake Console, si vous les avez ajoutés à la base de données et que vous avez activé l'option **Autoriser la connexion à l'IU**. Reportez-vous à [Gérer les utilisateurs de base de données](wxe1659392685092.md).
