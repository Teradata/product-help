Lorsque vous exécutez une tâche dans ModelOps, vous sélectionnez un pool de nœuds et configurez les ressources pour cette tâche. La sélection des ressources de calcul correctes dans ModelOps dépend du type de charge de travail, de la taille des données et des exigences de performances.

|Option|Description|Cas d'utilisation recommandés|
|-------|------------|----------------------|
|Calcul général|5 nœuds de 16 vCPU, mémoire : 64 Go de RAM|Idéal pour l'exploration, l'analyse et les fonctions dans la base de données|
|Mémoire élevée|3 nœuds de 64 vCPU, mémoire : 512 Go de RAM|Idéal pour les grands ensembles de données et les charges de travail utilisant beaucoup de mémoire|
|Accéléré par GPU|1 nœud équipé de 8xNVIDIA H200, 64 CPU, mémoire : 503 Go de RAM|NVIDIA H200 optimisé pour les services d'inférence|
