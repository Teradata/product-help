Puede computar estadísticas para sus características y objetivos al crear o editar una plantilla de conjunto de datos. Para poder realizar un seguimiento de la desviación de datos, es necesario almacenar la información estadística de las características.

1.  Consulte [Crear plantilla de conjunto de datos](frd1725409311264.md) para ver los pasos para acceder a la pestaña **Características**.


1.  En la pestaña **Características**, después de ejecutar la consulta SQL para recuperar las características, la tabla correspondiente muestra todas las disponibles. Seleccione **Validar metadatos**.

    ModelOps muestra una advertencia sobre los destinos que carecen de estadísticas y la opción de generar o volver a generar estadísticas para el destino seleccionado.


1.  Para generar metadatos, seleccione el tipo de característica para cada característica que desee incluir marcando la casilla que aparece junto a cada una en la tabla.


1.  Seleccione **Generar metadatos** para generar metadatos para las características que no los tengan.

    Seleccione **Volver a generar metadatos** para actualizar los metadatos existentes.


1.  Verifique los nombres de las características en el cuadro de diálogo y, a continuación, seleccione **Generar metadatos** o **Volver a generar metadatos**.

    Se inicia el proceso de computación de estadísticas.


1.  Seleccione el ![icono de Cerrar](Images/teg1680569591203.svg) cuando se haya completado el progreso de la computación de estadísticas.


1.  Cuando disponga de metadatos válidos, seleccione **Ver detalles** para consultar las estadísticas computadas.

    El mismo proceso puede aplicarse también al objetivo.


Para BYOM, se ejecuta un trabajo de computación de estadísticas después de importar el modelo. Para ejecutarlo manualmente para un modelo BYOM, haga clic con el botón secundario en una versión del modelo > **Estadísticas computacionales**.
