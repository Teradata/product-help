Un administrador de la organización puede gestionar la autorización de dispositivos OAuth 2.0. Esta opción controla el acceso a la REST API de Open Analytics Framework de la consola para todos los entornos aprovisionados en la organización. Si está deshabilitada, puede usar un token de acceso personal (PAT) para acceder a las REST API de Open Analytics.

En la página **Inicio**, seleccione el menú ![menú de tres puntos](Images/zsz1597101912145.svg) de OAuth para deshabilitar o habilitar OAuth 2.0.
