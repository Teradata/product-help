Use **Editor de SQL** para examinar objetos de base de datos y crear, editar, ejecutar y guardar secuencias de comandos y comandos SQL. El editor tiene funciones de análisis precargadas para consultar datos y un sencillo creador de consultas de apuntar y hacer clic.

En la barra de herramientas, seleccione **Editor de SQL** ![Icono de editor SQL](Images/swv1689722766775.png).
