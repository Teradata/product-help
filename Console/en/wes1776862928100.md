You can see the results of your query in a result sets tab. You can filter and sort your results in the following ways:

-   **Filtering**: Select the filter icon in the column header to engage filtering.


-   **Basic**: Select column values to filter.


-   **Advanced**: Select advanced criteria such as **Range** or **Contains** depending on the column type to filter.


**Sorting**: You can sort by columns in ascending or descending order. To sort by multiple columns, hold the **CMD** (for Mac) or **CTRL** (for Windows) button and click the sort icon in the column header of the additional rows.
