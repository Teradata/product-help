Teradata Enterprise Vector collection is a specialized database designed to store and manage high-dimensional vectors, that is, numerical representations of data like text and images. Rather than rely on exact keyword matches, vector collections use a semantic search by storing data as vectors. The key use cases include:

•Scalable search across large sets of structured or unstructured datasets

•Administrative management of vector collections

•Semantic search and retrieval

•AI-assisted content discovery

The key features are:

•Efficient storage and indexing of large vector datasets

•Integration with AI and machine learning workflows

•Enterprise-grade security, governance, and reliability

Note: You must be a database user to interact with vector collection. There are two vector collection roles, the vector collection admin role creates and modifies vector collection objects and the vector collection user role who enables the viewing of the vector collection objects.
