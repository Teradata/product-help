You can access Editor settings using **Editor** > ![](Images/xgv1772742212678.png) > **Settings,** to configure:

-   **Database objects to display**: By default, you can display only the objects you have privileges for. Enable **Show all objects** to display all objects.


-   **Error response**: When an error occurs, select one of the following:

    -   **Continue running the script**


    -   **Stop running the script**


    -   **Ask me every time** (Default)


-   **Results**: By default, when you run a script, your existing results are retained in a tab for later reference. To replace older results each time, disable **Create a new tab for each script run**.
