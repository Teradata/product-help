将数据上传到 Teradata Vantage
=============================

使用编辑器上传笔记本电脑或 PC 中的 CSV 数据，而不用运行 SQL 命令。

配合使用 NOS 和编辑器或 Jupyter 来运行 SQL 命令，将 CSV、JSON 或 Parquet 数据从外部库上传到本地表。
