Puede elegir Básico o Unstructured para su colección de vectores.

**Básico**: seleccione esta opción para una fragmentación liviana y rápida. El contenido se divide principalmente por tamaño y las reglas configurables.

Este método de fragmentación es mejor para documentos de texto simples con estructura predecible. Para documentos complejos, puede tener oraciones separadas que pueden generar pérdida de formato y afectar la precisión de los resultados. Puede corregir esto usando Unstructured.

**Nota:** La fragmentación básica tal vez no conserve la estructura semántica del documento.

**Unstructured**: seleccione Unstructured para una fragmentación que respete la estructura. Este método conserva los elementos del documento, como encabezados, tablas, listas y secciones.

Este método de fragmentación se recomienda para PDF, documentos enriquecidos y contenido complejo. Los fragmentos de mayor calidad producidos son útiles para cargas de trabajo de RAG y búsqueda semántica.
