Puede importar agentes desde archivos locales. Para ello, seleccione el archivo .zip requerido y luego cárguelo. Asegúrese de proporcionar los metadatos del agente antes de continuar.

**Nota:** Solo se permiten archivos .zip o hasta 100 MB.

A continuación, encontrará un ejemplo de un paquete y su estructura.

agent-package.zip

├── agent.py \# Required - Main agent entrypoint

├── config.yaml \# Required - Agent configuration

├── requirements.txt \# Required - Python dependencies

├── modules/ \# Optional - Additional Python modules

│ └── *.py

└── resources/ \# Optional - Additional resources

Si se produce un error, compruebe los detalles proporcionados y vuelva a intentarlo.

Entre los errores comunes, se incluyen servidor no disponible, credenciales incorrectas y problemas de acceso.
