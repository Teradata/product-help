Comparación entre notebooks de Jupyter y nativos de Teradata.

|Categoría|Notebooks nativos de Teradata|Jupyter Notebooks|
|---------|--------------------------|------------------|
|Kernel de SQL|Kernel de SQL nativo|Incluido|
|Navegación por base de datos|Navegador + Administrador de conexiones|Ninguno|
|Analíticas en la base de datos|Teradataml / tdplyr integrado|Requiere configuración manual|
|Visualización|Vega + Widgets de IU (no se requieren librerías de Python)|Matplotlib/seaborn/plotly|
|ML/BYOM UI|teradatamlwidgets (AutoML, almacén de características, almacén de vectores, BYOM)|Ninguno|
|Esfuerzo de configuración|Bajo|Alto|
|Mejor para|SQL de Teradata + Vantage Analytics + Flujos de trabajo empresariales|Ciencia de datos general|
