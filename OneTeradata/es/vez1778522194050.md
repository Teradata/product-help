Cree una clave virtual para darle a un usuario o aplicación acceso seguro a todos los modelos de IA a través de la puerta de enlace.

**Requisitos previos:**

-   Se registra al menos un modelo en el Centro de modelos de IA.


-   Tiene acceso administrativo.


-   Sabe si la clave es para una persona o una organización.


1.  Seleccione la pestaña **Claves virtuales** > **Crear clave virtual**.


1.  En la sección **Propiedad**, especifique quién es el dueño de la clave:

    -   **Propiedad de**

        Seleccione el tipo de propietario:

        **Usuario**: se asigna cuando un individuo necesita acceso directo a modelos de IA. Por ejemplo, un científico de datos que ejecuta experimentos.

        **Cuenta de servicio**: se asigna cuando una aplicación o sistema automatizado necesita acceso. Por ejemplo, un chatbot de producción.


    -   **ID de usuario**

        Introduzca la dirección de correo electrónico del usuario.


    -   **Etiquetas**

        Agregue etiquetas para categorizar el uso. Separe varias etiquetas con comas. Por ejemplo, gpt4, premium, producción.


1.  En la sección **Detalles de clave**, configure el acceso a los modelos:

    -   **Nombre de clave**

        Introduzca un nombre descriptivo para identificar esta clave. Por ejemplo, Clave de sistema de producción.


    -   **Modelos**

        Seleccione los modelos a los que puede acceder la clave. Para permitir el acceso a todos los modelos, seleccione Todos.


1.  En la sección **Caducidad**, establezca cuándo vence la clave y seleccione la hora.


1.  (Opcional) Para configurar los límites de velocidad, desactive **Usar los valores predeterminados** y actualice las opciones avanzadas.

    -   **Tokens por minuto (TPM)**

        Limita el número de tokens procesados por minuto. Tanto los tokens de entrada como los de salida se tienen en cuenta para este límite. Las solicitudes se restringen o rechazan cuando se supera el límite.


    -   **Solicitudes por minuto (RPM)**

        Limita el número de solicitudes de API por minuto. Cada solicitud se cuenta como una, independientemente del tamaño. Las solicitudes se rechazan cuando se supera el límite.


1.  Seleccione **Crear clave virtual**.

    **Importante**: La clave solo se muestra una vez. Cópiela de inmediato y compártala de forma segura con el usuario pretendido. Si se pierde, debe crear una nueva.
