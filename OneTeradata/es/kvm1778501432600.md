Acceda a detalles del endpoint necesarios para conectar sus aplicaciones a un modelo de IA. Use esta información para comprobar las implementaciones.

-   **Detalles del endpoint**

    Muestra la URL usada para enviar solicitudes a través de la puerta de enlace de su organización, el nombre de la puerta de enlace y el nombre del modelo público.


-   **Referencia de API**

    Proporciona ejemplos de código listo para usar para conectarse con el modelo, lo que incluye el método de autenticación y endpoint. Los ejemplos están disponibles en formatos como cURL y Python.
