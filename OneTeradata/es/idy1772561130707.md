Define la colección de vectores con un nombre único y proporciona una descripción adecuada.

Para crear una colección de vectores, debes seleccionar ya sea PDF desde tu almacenamiento local o información previamente fragmentada desde una tabla de base de datos o incrustaciones preexistentes (datos vectorizados) como tipo de dato.

Cuando termines, selecciona la ubicación de la base de datos para almacenar la nueva colección. De forma predeterminada, se usa la misma base de datos en la que iniciaste sesión.
