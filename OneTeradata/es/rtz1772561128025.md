Puedes crear y compartir documentos que contengan código activo, ecuaciones, visualizaciones y texto narrativo similares al uso de cuadernos Jupyter. Los cuadernos se usan en la ciencia de datos, el aprendizaje automático y la documentación técnica. Puedes:

-   Escribir y ejecutar código en fragmentaciones pequeñas


-   Usar varios lenguajes de programación (como Python)


-   Integrar gráficos, cuadros y planes usando bibliotecas (como Matplotlib, Seaborn o Plotly)


-   Combinar texto de marcado con código para crear flujos de trabajo legibles


-   Presentar resultados de modelo y análisis de datos


-   Crear tutoriales, guías técnicas y búsqueda reproducible


-   Habilitar el uso compartido y control de versiones para la validación y reproducción por parte del equipo


-   Usar en conjunto con plataformas de IA/ML (como TensorFlow, PyTorch y Teradata AI Factory)
