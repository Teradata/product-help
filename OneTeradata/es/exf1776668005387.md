Cuando seleccione el despliegue requerido, podrá ver sus detalles en diversas pestañas.

**Información general**: vea los detalles del despliegue, como detalles del endpoint y JSON de configuración, entre otros.

**Referencia de API**: vea el código de API en Python, Shell y LangChain.

Podemos agregar un usuario como colaborador en un despliegue en particular.

Ese usuario luego puede administrar el despliegue, igual que la persona que lo creó

**Usuarios**: puede agregar un usuario como colaborador en un despliegue en particular. Para ello use + y el ID de usuario, y confirme su creación. El usuario recientemente agregado ahora puede administrar el despliegue, igual que la persona que lo creó.

De igual manera, puede eliminar al usuario seleccionando Eliminar y confirmar. Cuando se apliquen los cambios, podrá ver la lista revisada de usuarios para el despliegue seleccionado.

**Registros**: vea los registros del despliegue, como la hora de creación y los estados de los diversos procesos, entre otros. Si fuera necesario, puede descargar los registros.
