Selecciona el modelo de incrustación requerido que deseas usar. Los modelos de AWS disponibles son:

· Modelo de incrustaciones de texto de Amazon Titan V2

· Modelo de incrustaciones de texto de Amazon Titan V1

· Modelo de incrustaciones de imagen de Amazon Titan V1

**Nota**: Al cambiar el modelo de incrustación, se volverán a incrustar todos los datos existentes en la colección de vectores y se incrustará de forma simultánea cualquier nuevo dato.

También puedes usar modelos de Azure que estén disponibles.

· Tex Embedding Ada 002

· Tex Embedding 3 Small

· Tex Embedding 3 Large

Además, puedes seleccionar **gpt-4.1-mini** para el modelo de lenguaje (conversación).
