Una celda es un contenedor para código o texto dentro de un notebook. Contiene el código ejecutable (Python, Teradata SQL) y el texto de marcado para explicaciones o documentación.

|Tipo de celda|Propósito|Comportamiento|
|----------|--------|---------|
|Celda de código|Escriba y ejecute código Python|El kernel ejecuta el código Python y muestra el resultado|
|Celda de marcado|Agregue documentación y texto con formato|Convierte la sintaxis de marcado en texto con estilo|
|SQL|Escriba y ejecute código de Teradata SQL|El kernel ejecuta el código SQL y muestra el resultado|
