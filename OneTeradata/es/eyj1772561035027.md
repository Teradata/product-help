Como administrador o analista de datos, puede administrar las colecciones de vectores que haya creado o a las que haya tenido acceso.

En esta página, puede buscar colecciones activas usando diversos parámetros, como nombre de colección, descripción, tipo de dato, etc.

**Nota:** AI Studio muestra las colecciones de vectores creadas en VantageCloud Enterprise 3.2 o usando posteriormente API de la V2. Las colecciones creadas con API anteriores (V1), como las de VantageCloud Enterprise 3.1, no se mostrarán, ni siquiera si actualiza a 3.2.

También puede aplicar filtros para restringir listas grandes a fin de encontrar elementos relevantes con rapidez.

Según los privilegios de su cuenta, puede hacer lo siguiente:

Crear

Eliminar

Agregar datos

Ver detalles y luego editarlos

Volver a intentar una colección de vectores errónea después de comprobar el error correspondiente

**Estados**

A continuación se encuentran los diversos estados:

**Lista**: colección de vectores lista para usar.

**Creando**: creación en curso para la colección de vectores.

**Error**: error al crear la colección de vectores. Mira los detalles del error y haga las correcciones que sean necesarias antes de volver a intentarlo.

Puedes organizar las colecciones de vectores según el estado.
