Como administrador o analista de datos, puedes administrar las colecciones de vectores que hayas creado o a las que hayas tenido acceso.

En esta página, puedes buscar colecciones activas usando diversos parámetros, como nombre de colección, descripción, tipo de dato, etc.

También puedes aplicar filtros para restringir listas grandes a fin de encontrar elementos relevantes con rapidez.

Según los privilegios de tu cuenta, puedes:

Crear

Eliminar

Agregar datos

Ver detalles y luego editarlos

Volver a intentar una colección de vectores errónea después de comprobar el error correspondiente

**Estados**

A continuación se encuentran los diversos estados:

**Lista**: colección de vectores lista para usar.

**Creando**: creación en curso para la colección de vectores.

**Error**: error al crear la colección de vectores. Mira los detalles del error y haz las correcciones que sean necesarias antes de volver a intentarlo.

Puedes organizar las colecciones de vectores según el estado.
