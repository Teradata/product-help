Analice, busque y compare modelos de IA registrados en un centro de modelos de IA. Use los detalles del modelo para evaluar las capacidades e identificar opciones que se ajusten a sus flujos de trabajo.

## Qué muestra la página


-   **Nombre**

    Muestra el nombre del modelo.


-   **Alias o ID de modelo**

    Muestra el identificador exclusivo o alias asignado al perfil del modelo.


-   **Versión**

    Muestra el número de versión del modelo si está disponible.


-   **Tipo**

    Muestra la categoría del modelo. Por ejemplo, incrustación para modelos de incrustación de texto o imagen, o conversación para modelos conversacionales.


-   **Proveedor**

    Muestra el proveedor de infraestructuras que aloja el modelo. Por ejemplo, bedrock.


-   **Descripción**

    Muestra una breve descripción del modelo.


-   **URL base de la API**

    Muestra la URL base para el endpoint de API de modelo si está configurada.


-   **Longitud del contexto**

    Muestra el número máximo de tokens que el modelo puede procesar en una sola solicitud si se especifica.


-   **Límite de tasa**

    Muestra el número máximo de solicitudes permitido dentro de un plazo definido si se especifica.


-   **Actualizado a las**

    Muestra la fecha y hora en que la entrada del modelo se actualizó por última vez.


## Quién hace qué


-   **Administrador**

    Vea todos los modelos registrados.


-   **Usuario**

    Vea solo los modelos asignados a su clave virtual. Para solicitar acceso a modelos adicionales, póngase en contacto con su administrador.
