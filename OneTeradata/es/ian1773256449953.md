La página de inicio de Teradata AI Studio Enterprise MCP proporciona una guía rápida y dentro del producto que lo ayuda a conectarse con el servidor Teradata Enterprise MCP y usarlo. Use la página de inicio para obtener información de conexión, generar tokens de autenticación y acceder a ejemplos de integración.

En ella se proporciona la siguiente información. Para detalles sobre cómo trabajar con herramientas de MCP y Teradata Enterprise MCP, consulte la [Guía del usuario de Teradata Enterprise MCP](https://docs.teradata.com/access?resourceid=enterprise-mcp-book).

-   **Endpoint de servidor**

    Agregue el endpoint de servidor MCP a su notebook o configuración de herramientas externas para conectarse con el servidor Enterprise MCP.

    -   **Servidor MCP predeterminado**

        Use la URL del servidor mostrada en la IU para conectarse con el servidor MCP predeterminado.


    -   **Servidor MCP desplegado por el administrador**

        Recupere la URL del endpoint del servidor usando la API **Enumerar todos los servidores**. Consulte [Enumerar todos los servidores](https://docs.teradata.com/access?resourceid=mcp-srvr-mg-list-servers&utm_source=console&utm_medium=iph) para más detalles.

    Debe conectarse con la red privada donde se despliega AI Studio para acceder a este endpoint.


-   **Generar un token**

    Necesitará un token para autenticar las solicitudes. Seleccione un método de autenticación. Para más información, consulte [Autorización, RBAC y autenticación](https://docs.teradata.com/access?resourceid=author-rbac-authen).


-   **Conectar clientes externos**

    Conecte clientes de MCP externos al servidor para ampliar el acceso MCP a recursos y herramientas locales. Para más información, consulte [Conectar clientes de MCP al servidor Enterprise MCP](https://docs.teradata.com/access?resourceid=mcp-connect-agent).
