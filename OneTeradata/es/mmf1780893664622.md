El explorador de datos le permite explorar sus bases de datos de Teradata y sus objetos directamente desde la barra lateral del notebook.

Puede hacer lo siguiente:

• Expandir una base de datos para ver sus objetos organizados por tipo

• Expandir una tabla o vista para ver sus columnas junto con sus tipos de datos

• Buscar: filtrar el árbol de la base de datos en tiempo real para encontrar objetos por nombre

• Destacar/Anular destaque: marcar una base de datos como destacada para anclarla a la parte superior del panel para acceso rápido

• Actualizar: recargar el último estado de una base de datos, tabla o vista
