Fragmentación es el proceso de desarmar documentos grandes o datos en pedazos más pequeños, manejables y semánticamente significativos llamados **fragmentos**.

Selecciona **Básico** para documentos simples o **Unstructured.io** para documentos complejos con tablas e imágenes.

**Básico**: métodos de división recursivos, de tamaño fijo y simples.

**Unstructured.io**: selecciona si tus documentos son datos multimodales, como texto, PDF, imágenes o audio, y desestructurados.

De forma predeterminada, cada fragmento tiene 512 caracteres. Si fuera necesario, puedes cambiar este tamaño de fragmento.

Al cambiarlo, puedes también seleccionar el encabezado o pie o hasta 1000 píxeles.

**Nota**: Al seleccionar el encabezado y el pie se agregan a los datos que se procesan.
