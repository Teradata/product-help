La fragmentación es el proceso de descomponer documentos o datos grandes en partes más pequeñas y administrables, que son significativas en términos semánticos, denominadas **fragmentos**.

Seleccione **Básica** o **Unstructured**.

**Básica**: úselo para una fragmentación rápida basada en tamaño para documentos de texto simples. Este proceso tal vez no conserve la estructura del documento.

**Unstructured**: úselo para una fragmentación que respeta la estructura y conserva los encabezados, las tablas y las secciones de los documentos para una calidad de recuperación mejorada.

De forma predeterminada, cada fragmento contiene 512 caracteres. Si fuera necesario, puede cambiar este tamaño de fragmento.

Al cambiarlo, puede también seleccionar el encabezado o pie o hasta 1000 píxeles.

**Nota**: Al seleccionar el encabezado y el pie se agregan a los datos que se procesan.
