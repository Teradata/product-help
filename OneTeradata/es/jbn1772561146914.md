Al crear una colección de vectores, puedes seleccionar un lenguaje en los modelos de AWS disponibles o diferirlo para más tarde.

Si seleccionas AWS, puedes elegir entre Anthropic Claude 3 Haiku y Anthropic Claude 3.5 Sonnet.

Selecciona Anthropic Claude 3 Haiku para tareas de volumen alto y baja latencia en las que la velocidad y economía son cruciales, como un análisis rápido de datos, chatbots de servicio al cliente instantáneos, moderación de contenido, generación de sugerencias de código rápido y procesamiento de grandes conjuntos de documentos para una extracción de metadatos y categorización rápidas, especialmente cuando necesites un razonamiento rápido y confiable, que no necesariamente sea más complejo. Es excelente para tareas que requieren tiempos rápidos de respuesta y una manipulación de conjuntos de datos grandes de forma rápida, lo que lo hace ideal para aplicaciones orientadas a los usuarios y procesamiento de datos de backend.

Selecciona Anthropic Claude 3.5 Sonnet para una potente combinación de velocidad, economía y rápido desempeño en codificación, análisis de datos y flujos de trabajo complejos, especialmente cuando necesites una iteración rápida, manejes grandes contextos o tengas que entender visualmente sin el coste de los modelos de primer nivel como Opus. Es ideal para desarrolladores, científicos de datos y negocios que necesitan IA escalable para atención al cliente, migración de código y creación de contenido con matiz humano.
