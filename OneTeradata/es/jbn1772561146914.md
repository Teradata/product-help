Al crear una colección de vectores, puede seleccionar un modelo de lenguaje en los modelos de AWS disponibles o diferirlo para más tarde.

Si selecciona AWS, puede elegir entre Anthropic Claude 3 Haiku y Anthropic Claude 3.5 Sonnet.

Si selecciona Azure, puede usar GPT-4.1-mini para el modelo de lenguaje (conversación).

Seleccione Anthropic Claude 3 Haiku para tareas de volumen alto y baja latencia en las que la velocidad y economía sean cruciales, como un análisis rápido de datos, chatbots de servicio al cliente instantáneos, moderación de contenido, generación de sugerencias de código rápido y procesamiento de grandes conjuntos de documentos para una extracción de metadatos y categorización rápidas, especialmente cuando necesite un razonamiento rápido y confiable, que no necesariamente sea más complejo. Es excelente para tareas que requieren tiempos rápidos de respuesta y una manipulación de conjuntos de datos grandes de forma rápida, lo que lo hace ideal para aplicaciones orientadas a los usuarios y procesamiento de datos de backend.

Seleccione Anthropic Claude 3.5 Sonnet para una potente combinación de velocidad, economía y rápido desempeño en codificación, análisis de datos y flujos de trabajo complejos, especialmente cuando necesite una iteración rápida, maneje grandes contextos o tenga que entender visualmente sin el coste de los modelos de primer nivel como Opus. Es ideal para desarrolladores, científicos de datos y negocios que necesitan IA escalable para atención al cliente, migración de código y creación de contenido con matiz humano.

Seleccione GPT-4.1 Mini, si necesita un modelo de IA de baja latencia y alta economía como una alternativa más pequeña a los sistemas de clase GPT-4. Es ideal para aplicaciones de alto volumen y tiempo real, y admite una **ventana de contexto de token de 1 millón** excepcionalmente grande, lo que permite un análisis de un solo paso de documentos largos o bases de código grandes. Con soporte de entrada multimodal y un fuerte rendimiento en codificación y razonamiento, GPT 4.1 Mini es adecuado para agentes, herramientas de desarrollador y entornos sensibles al coste de escala de producción.
