Los modelos de lenguaje (también conocidos como modelos de conversación) se usan para crear una respuesta de lenguaje natural durante la recuperación. Al hacer una pregunta de lenguaje natural a los datos, el modelo de lenguaje usa los resultados de Top-K devueltos por el algoritmo de búsqueda eficiente y da una respuesta de lenguaje natural precisa.

Puedes seleccionar Modelo de AWS o Modelo de Azure, o hacer la selección más tarde. Si seleccionas Modelo de AWS, puedes elegir entre **Anthropic Claude 3 Haiku** o **Anthropic Claude 3.5 Sonnet**. Si seleccionas Modelo de Azure, puedes seleccionar entre **text-embedding-ada-002** o **text-embedding-3-small** o **text-embedding-3-large** para la incrustación. Además, puedes seleccionar **gpt-4.1-mini** para el modelo de lenguaje (conversación).
