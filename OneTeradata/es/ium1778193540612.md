Un kernel es un motor o intérprete que ejecuta el código dentro de su notebook. Cada kernel en notebooks nativos de Teradata admite los siguientes lenguajes:

-   Python


-   Teradata SQL


Dentro de un kernel, puede hacer lo siguiente:

-   Ejecutar código desde su notebook


-   Administrar variables y llevar un seguimiento del estado de la sesión


-   Generar resultados después de ejecutar su código


Cuando ejecuta una celda, el kernel interpreta el código, lo ejecuta y envía el resultado de regreso a su notebook.
