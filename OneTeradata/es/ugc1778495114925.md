El Centro de modelos de IA le permite acceder, administrar y supervisar modelos fundacionales aprobados para usarlos en su organización.

Use la pestaña **Descripción general** para escanear los modelos fundacionales aprobados para su organización. Compare las capacidades, revise los límites de token e identifique modelos que se ajusten a sus flujos de trabajo.

**Tarjetas de modelo**

Revise una instantánea de nivel alto de cada modelo para comparar capacidades de un vistazo.

**Límites de token**

Use información de token en cada tarjeta para comprender cuánta entrada acepta un modelo y cuánta salida genera.

-   **Tokens de entrada máximos**

    El texto máximo que puede enviar en una sola solicitud. Define la ventana de contexto del modelo.


-   **Tokens de salida máximos**

    El texto máximo que el modelo puede generar en una respuesta.


## Quién hace qué


-   **Administrador**

-   **Usuario**

    Introduzca su clave virtual para ver los modelos que tiene asignados.
