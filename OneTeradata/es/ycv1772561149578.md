El almacén de vectores de Teradata Enterprise es una base de datos especializada diseñada para almacenar y administrar vectores de gran dimensión, es decir, representaciones numéricas de los datos como texto e imágenes. Almacenar datos como vectores permite a los usuarios de datos consultar la colección con búsqueda semántica, búsqueda híbrida y también coincidencias exactas de palabras clave.

Los casos de uso clave incluyen:

• Búsqueda escalable entre grandes conjuntos de datos estructurados y datos no estructurados

• Gestión administrativa de colecciones de vectores

• Búsqueda y recuperación semántica

• Detección de contenido asistida por IA

Las características clave son:

• Almacenamiento e indexación eficientes de grandes conjuntos de datos de vectores

• Integración con flujos de trabajo de aprendizaje automático e IA

• Confiabilidad, gobernanza y seguridad de grado empresarial

**Nota**: Debe ser un usuario de base de datos para interactuar con la colección de vectores. Hay dos roles en la colección de vectores, administrador y usuario. El administrador de la colección de vectores otorga a los usuarios diversos privilegios, como crear, administrar o usar las colecciones de vectores.
