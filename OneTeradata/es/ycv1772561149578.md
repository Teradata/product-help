La colección de vectores de Teradata Enterprise es una base de datos especializada diseñada para almacenar y administrar vectores de dimensiones, es decir, representaciones numéricas de datos como texto e imágenes. En lugar de confiar en coincidencias exactas de palabras clave, las colecciones de vectores usan una búsqueda semántica mediante el almacenamiento de datos como vectores. Los casos de uso clave incluyen:

• Búsqueda escalable entre grandes conjuntos de datos estructurados y no estructurados.

• Gestión administrativa de colecciones de vectores

• Búsqueda y recuperación semántica

• Detección de contenido asistida por IA

Las funciones clave son:

• Almacenamiento e indexación eficientes de grandes conjuntos de datos de vectores

• Integración con flujos de trabajo de aprendizaje automático e IA

• Confiabilidad, gobernanza y seguridad de grado empresarial

**Nota**: Debes ser un usuario de base de datos para interactuar con la colección de vectores. Hay dos funciones de colección de vectores. La de administrador crea y modifica objetos de colección de vectores y la de usuario, permite ver dichos objetos.
