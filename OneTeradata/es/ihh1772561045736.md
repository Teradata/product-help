Operaciones de aprendizaje automático (MLOps) es una metodología que unifica el desarrollo del modelo de aprendizaje automático con la administración y despliegue operativo. ModelOps es un subconjunto de MLOps que se centra en el despliegue, la supervisión y la gestión de modelos de aprendizaje automático en entornos de producción. Cierra la brecha entre el desarrollo de modelos y el uso operativo, lo que garantiza que los modelos se integren de forma confiable en aplicaciones y procesos comerciales.

ModelOps de AI Studio administra la operacionalización de las analíticas avanzadas en Teradata Vantage.

Además, AI Studio ModelOps ofrece una interfaz de usuario web fácil de usar, una interfaz de línea de comandos, un kit de desarrollo de software (SDK) R o Python para desplegar u operacionalizar modelos analíticos avanzados en Teradata Vantage usando el proceso Analytics 1‑2‑3.

## Analytics 1‑2‑3


Analytics 1‑2‑3 es un enfoque que le ayuda a preparar los datos, entrenar modelos y desplegar modelos en la plataforma de Vantage. El proceso incluye tres etapas:

**Preparación de datos**

Use la plataforma de Teradata para preparar los datos de VantageCloud Enterprise para el modelado. Conecte las fuentes de datos de VantageCloud Enterprise y aplíquelas en las funciones de base de datos para integración, exploración, limpieza e ingeniería de características.

**Entrenar el modelo**

Use las herramientas, los lenguajes y las bibliotecas que utiliza con frecuencia. Puede entrenar modelos usando marcos como SAS, scikit-learn, XGBoost, MLlib o H2O.

**Desplegar el modelo**

Use ModelOps para traer su propio modelo (BYOM) entrenado externamente a Vantage o hacer un seguimiento del proceso de entrenamiento. El modelo desplegado se ejecuta en Vantage para generar predicciones y se monitorea a lo largo de su ciclo de vida.
