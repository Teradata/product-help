To deploy an agent, select the agent to deploy from the agent registry.

The following are the steps:

**Agent Details**: Select from the available agents to deploy. Once selected, all the available versions are shown. Select the required one to view the details.

Select **Next**.

**Metadata**: Select the metadata and provide the name and description.

**Configure endpoints**: View the respective JSON file. If required, you can edit it.

**Resources**: View the Resource settings and configure resources. In Configure Resources, you can choose either Standard or Custom.

**Summary**: View the summary of the agent you want to deploy.

**Deploy** to deploy the agent. You can view the progress, status, and other details.

Note: You don't need to import when deploying the agents. You import only when adding an agent to the registry, and creating more versions.
