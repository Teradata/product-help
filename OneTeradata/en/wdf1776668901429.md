To deploy agent, you can import from the existing ones.

The following are the steps involved.

**Agent Details**: Select from the available agents to deploy. Once selected, if any versions are available, they'll show. Select the required one. You'll see all the details. Select **Next**.

**Metadata**: Select the metadata. Provide the name, description, etc.

**Configure endpoints**: View the respective JSON file. If required, you can edit it.

**Resources**: View the Resource settings and configure resources. In Configure Resources, you can choose either Standard or Custom.

**Summary**: View the summary of the agent you want to deploy.

Select **Deploy** to deploy the agent. You can view the progress, status, and other details.
