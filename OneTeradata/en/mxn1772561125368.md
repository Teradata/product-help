Tera is your chat-based AI assistant to simplify how you interact with AI Studio. Use Tera to discover datasets, understand schemas, inspect patterns in data, and generate basic visualizations by using natural language in AI Studio.

Tera provides read-only access to available metadata and metrics within your existing data access permissions.

Tera does not modify system configuration or replace administrative or operational monitoring tools.

## What you can do with Tera


-   Explore and analyze data from Teradata tables


-   Inspect distributions, trends, and anomalies


-   List accessible databases and view table schemas


-   Review column definitions and data types


-   View database performance indicators and table statistics, where permitted


-   Generate line, pie, radar, and polar charts from query results


-   Get guidance on Teradata functionality and data discovery workflows
