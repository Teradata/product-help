Comparison between Jupyter notebooks and Teradata native notebooks.

|Category|Teradata native notebooks|Jupyter notebooks|
|---------|--------------------------|------------------|
|SQL Kernel|Native SQL Kernel|Included|
|Database navigation|Navigator + Connection Manager|None|
|In-database analytics|Teradataml / tdplyr built in|Requires manual setup|
|Visualization|Vega + UI widgets (no Python libs required)|Matplotlib/seaborn/plotly|
|ML/BYOM UI|teradatamlwidgets (AutoML, Feature Store, Vector Store, BYOM)|None|
|Setup effort|Low|High|
|Best for|Teradata SQL + Vantage analytics + enterprise workflows|General data science|
