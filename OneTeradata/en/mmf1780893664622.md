Data explorer allows you to browse your Teradata databases and their objects directly from the notebook sidebar.

You can do the following:

• Expand any database to see its objects organized by type

• Expand a table or view to see its columns along with their data types

• Search: filter the database tree in real time to find objects by name

• Star / Unstar: mark a database as starred to pin it to the top of the panel for quick access

• Refresh: reload the latest state of a database, table, or view
