AI Model Hub enables you to access, manage, and monitor foundation models approved for use in your organization.

Use the **Overview** tab to scan foundation models approved for your organization. Compare capabilities, review token limits, and identify models that fit your workflows.

**Model cards**

Review a high-level snapshot of each model to compare capabilities at a glance.

**Token limits**

Use token information on each card to understand how much input a model accepts and how much output it generates.

-   **Max input tokens**

    The maximum text you can send in a single request. This defines the model's context window.


-   **Max output tokens**

    The maximum text the model can generate in one response.


## Who does what


-   **Administrator**

    You can add a model from this page. See [Add a model](fdy1778502087907.md).


-   **User**

    Enter your virtual key to view models assigned to you.
