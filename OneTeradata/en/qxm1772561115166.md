Use Tera to explore data through natural language prompts. You can access Tera either from the home page or by selecting **Tera** in the left navigation panel.

## Write effective prompts


Use the following guidance to get better results:

-   Start with a clear goal.


-   Provide relevant context.


-   Use specific keywords and details.


-   Break multi-part requests into smaller parts.


-   Refine the conversation by using follow-up prompts.


Example prompts:

-   List all databases.


-   Analyze the data.


-   Describe the columns in a table.


-   Show the tables related to customer churn.


-   Summarize the schema for the customer activity table.


-   Identify columns related to account status and transaction date.


-   Show trends in monthly churn for the banking dataset.
