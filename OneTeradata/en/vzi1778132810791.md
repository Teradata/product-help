Use standard settings or customize allocation to commit additional resources.

1.  On **Configure resources**, select additional resources.

    1.  Standard: Use this for mostly used everyday notebooks and light data exploration.

        1 vCPU, 2 GB memory


    1.  Custom: Select your resources: vCPU and memory (in GB). Adjust vCPU (1–12) and memory (2–51 GB) to match heavier workloads.

        Up to 16 vCPU, up to 64 GB memory


1.  Select **Start session**.
