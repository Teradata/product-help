You can import agents from GIT Repository. You must provide username and password as credentials, provide Agent Metadata, test the connection. Upon successful connection without errors, import the agents by selecting Import.

The following is an example of a package and its structure.

agent-package

├── agent.py \# Required - Main agent entrypoint

├── config.yaml \# Required - Agent configuration

├── requirements.txt \# Required - Python dependencies

├── modules/ \# Optional - Additional Python modules

│ └── *.py

└── resources/ \# Optional - Additional resources

If an error occurs, check the details provided, and retry.
