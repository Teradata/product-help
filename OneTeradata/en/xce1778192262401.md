A cell is a container for code or text within your notebook. It holds executable code (Python, Teradata SQL) and markdown text for documentation or explanations.

|Cell type|Purpose|Behavior|
|----------|--------|---------|
|Code cell|Write and run Python code|Kernel runs the Python code and displays output|
|Markdown cell|Add documentation and formatted text|Renders markdown syntax into styled text|
|SQL|Write and run Teradata SQL code|Kernel runs SQL code and displays output|
