The Teradata AI Studio Enterprise MCP home page provides quick, in-product guidance to help you connect to and use the Teradata Enterprise MCP server. Use the home page to get connection information, generate authentication tokens, and access integration examples.

The home page provides the following information. For details on working with Teradata Enterprise MCP and MCP tools, see the [Teradata Enterprise MCP User Guide](https://docs.teradata.com/access?resourceid=enterprise-mcp-book).

-   **Server endpoint**

    Add the MCP server endpoint to your notebook or external tool configuration to connect to the Enterprise MCP server.

    You must be connected to the private network where AI Studio is deployed to access this endpoint.


-   **Generate a token**

    You will need a token to authenticate requests. Select an authentication method. For more information, see [Authorization, RBAC, and Authentication](https://docs.teradata.com/access?resourceid=author-rbac-authen).


-   **Connect external clients**

    Connect external MCP clients to the server to extend MCP access to local tools and resources. For more information, see [Connect MCP Clients to the Enterprise MCP Server](https://docs.teradata.com/access?resourceid=mcp-connect-agent).
