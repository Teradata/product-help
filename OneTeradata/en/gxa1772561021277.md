Clear, structured prompts help Tera return more useful results. For the prompting best practices, see [Get started with Tera](qxm1772561115166.md).

## Start a new chat


Select **New chat** to start a new conversation. Starting a new chat clears the current conversation context and does not retain previous prompts or responses.

If you need continuity, copy the relevant responses before you start a new chat and include that context in the new conversation.

## Export charts


Use the Export button to download charts that you created or viewed in Tera. The button appears when a chart in generated.

**Supported chart formats**

-   **SVG** - Use for scalable chart images in presentations, documents, or other visual assets.


-   **HTML** - Use when you want to preserve interactivity.
