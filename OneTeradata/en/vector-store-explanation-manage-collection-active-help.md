As a data analyst or administrator, you can manage the vector collections that you have created or have access to. 

In this page you can search for active collections using various parameters, such as, collection name, description, data type, and so on.

You can also apply filters to narrow down large lists to find relevant items quickly.

Based on your account privileges, you can:

- Create
- Delete
- Add data
- View Details and then edit
- Re-attempt a failed vector collection after verifying the corresponding error

The following are the various statues:

- Ready: Ready to use vector collection.
- Creating: Creation in progress for the vector collection.
- Failed: The vector collection creation failed with error. View the error details and make necessary correction before retrying.
