Scan, search, and compare AI models registered in AI Model Hub. Use the model details to evaluate capabilities and identify options that fit your workflows.

## What the page shows


-   **Name**

    Shows the name of the model.


-   **Model ID or alias**

    Shows the unique identifier or alias assigned to the model profile.


-   **Version**

    Shows the version number of the model, if available.


-   **Type**

    Shows the category of the model. For example, embedding for text or image embedding models, or chat for conversational models.


-   **Provider**

    Shows the infrastructure provider that hosts the model. For example, bedrock.


-   **Description**

    Shows a brief description of the model.


-   **API base URL**

    Shows the base URL for the model API endpoint, if configured.


-   **Context length**

    Shows the maximum number of tokens the model can process in a single request, if specified.


-   **Rate limit**

    Shows the maximum number of requests allowed within a defined time period, if specified.


-   **Updated at**

    Shows the date and time when the model entry was last updated.


## Who does what


-   **Administrator**

    View all registered models.


-   **User**

    View only models assigned to your virtual key. To request access to additional models, contact your administrator.
