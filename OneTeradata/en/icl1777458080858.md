AgentOps within AI Studio is designed to manage, deploy, and monitor AI agents. It provides a centralized dashboard to track deployed agents, manage agent versions, and configure deployments across environments.

The following are main use cases of AgentOps.

Memory-Enabled Agents: Agents can maintain conversation history and context across sessions.

Teradata Database Integration: Build AI agents that can query and analyze data in Teradata databases.

Automate multi-step business processes using AI agents: Agents that can gather information and make recommendations.
