File explorer allows you to browse, manage, and organize your files and folders. The following table shows the folder structure used to organize the demo notebooks, public, and user spaces:

|Item|Description|
|-----|------------|
|Demo|Contains Teradata Notebooks demonstrations from ClearScape Analytics Experience|
|Teradata|Stores configuration, kernel information, log, and result files|

You can do the following:

-   Search interaction


-   Context menu: Open, download, cut, copy, paste, rename, copy file path, star, new folder, new file, delete


-   Upload, select file to upload


-   Create new file


-   Cancel upload, cancel all uploads


-   Unsaved field


-   Fullscreen
