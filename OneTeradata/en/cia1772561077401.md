Select the required embedding model that you want to use. The available AWS models are:

· Amazon Titan Text Embeddings V2 model

· Amazon Titan Text Embeddings V1 model

· Amazon Titan Image Embeddings V1 model

**Note**: Changing the embedding model will re-embed all existing data in the vector collection and simultaneously embeds any new data.

You can also use Azure models that are available.

· Tex Embedding Ada 002

· Tex Embedding 3 Small

· Tex Embedding 3 Large
