Once you select the required deployment, you can view its details in various tabs.

**Overview**: View the details of the deployment such as endpoint details, configuration JSON, and others.

**API Reference**: View the API code in Python, Shell, and LangChain.

**Users**: You can create new users or delete an existing user by selecting + and Delete icons. When you select + , enter the user ID and confirm user creation. Similarly, select Delete and confirm the user deletion. After confirming the change, you can view the revised list of users for the selected deployment.

**Logs**: View the logs of the deployment, including creation time, statuses of various processes, and so on. If required, you can download the logs.
