Once you select the required deployment, you can view its details in various tabs.

**Overview**: View the details of the deployment such as endpoint details, configuration JSON, and others.

**API Reference**: View the API code in Python, Shell, and LangChain.

we can add a user as collaborator on a particular deployment.

That user can then manage the deployment, similar to the person who created the deployment

**Users**: You can add a user as a collborator on a particular deployment by using + and provding the user ID and confirming the user creation. The newly added user can now manage the deployment similar to the user who created the deployment.

Similarly, delete the user by selecting Delete and confirm. Once the changes are applied, you can view the revised list of users for the selected deployment.

**Logs**: View the logs of the deployment, including creation time, statuses of various processes, and so on. If required, you can download the logs.
