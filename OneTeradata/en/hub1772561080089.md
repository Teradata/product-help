You can select various types of raw data as data source and convert it into vector embeddings using a large language model (embedding model).

You can upload data either from local files or database tables.

**Local files**

Select if you plan to upload text, documents, structured, or tablular data from your machine.

These files can include plain text files, articles, books, user reviews, customer feedback, and internal documentation which are often used for semantic search and Retrieval Augmented Generation (RAG) applications in AI.

You can drag and drop or browse files from your local machine.

**Note**: You can upload up to 10 files (100 MB limit) with the same schema.

**Database tables**

Select if you want to specify the database table and columns. Based on your selection of either Chunks only or Chunks and embeddings, you'll see various selections of tables, columns, and embeddings.
