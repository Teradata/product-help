You can select various types of raw data as data sources and convert into vector embeddings using a large language model (embedding model).

Text and Documents: This includes plain text files, articles, books, user reviews, customer feedback, and internal documentation (e.g., from Confluence or Jira). These are often used for semantic search and Retrieval Augmented Generation (RAG) applications in AI.

Structured or Tabular Data: Numerical features from databases (like user preferences, product characteristics, or transaction data) can be converted into vectors for recommendation systems, fraud detection, and anomaly detection.

You can drag and drop or browse files from your local machine.

**Note**: You can upload up to 10 files (100 MB limit) with the same schema.

Database tables: Select if you want to specify the database table and columns. Based on your selection of either Chunks only or Chunks and embeddings, you'll see various selections of tables, columns, and embeddings.

Unstructured: Select if you want to ingest large datasets from the Unstructured pipeline. With Unstrcutured.io, you can select images, text, and over 65 different file types as data.
