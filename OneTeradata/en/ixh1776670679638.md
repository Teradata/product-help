View the metrics and owner details for a deployment using the Observability tab.

Select the deployment to view various tabs.

**Dashboard**: The summary cards show the following:

- Total traces (requests processed)

- Token count (total tokens consumed)

- P50 latency (median response time)

- Error rate

**Traces**: View the versions available, status, etc. You can apply filters to refine the traces search. You can run the agent and view the input/output and corresponding metadata.

The details shown in the Traces tab are:

- Duration per request.

- Input/output tokens

- Error details for failed requests

- Spans showing internal agent steps

**Span**: Represents an individual operation within a trace.

**Metrics**: Represents quantitative measurements used to assess model performance.
