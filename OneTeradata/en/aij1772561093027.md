You can create vector collections from different types of data:

**Text and Documents**: This includes plain text files, articles, books, user reviews, customer feedback, images, and internal documentation (for example from Confluence or Jira). These are often used for semantic search and Retrieval-Augmented Generation (RAG) applications in AI.

**Structured or Tabular Data**: Numerical features from databases (like user preferences, product characteristics, or transaction data) can be converted into vectors for recommendation systems, fraud detection, and anomaly detection.
