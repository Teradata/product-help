Use standard settings or customize allocation to commit additional resources.

1.  On **Configure resources**, select additional resources.

    1.  Standard.

        1 GPU 2 vCPU, 16 GB memory


    1.  Custom. Select your resources: vCPU and memory (in GB).

        Up to 8 GPU, Up to 64 vCPU, up to 503 GB memory


1.  Select **Start session**.
