You can add new users or search the existing users, from this page. Using filters, you can search users based on name and access levels.

Select + to add new users to the selected vector collection to allow them to use the same. While creating the user, select the access level as Limited or Full.

- Limited: Provides access the vector collection with existing settings and test it in playground.
- Full: Provides access to modify the vector collection settings and test it in playground.

Note: You can only add users available in your environment.

Based on your privileges, you can also modify the access or delete the user from the vector collection.
