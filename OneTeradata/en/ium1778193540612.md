A Kernel is an engine or interpreter that runs the code inside your notebook. Each kernel on Teradata native notebooks supports the following languages:

-   Python


-   Teradata SQL


Within a Kernel you can do the following:

-   Run code from your notebook


-   Manage variables and keep track of the session state


-   Generate output after running your code


When you run a cell, the Kernel interprets the code, runs it, and sends the result back to your notebook.
