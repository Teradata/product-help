Vous pouvez sélectionner différents types de données brutes comme source de données et les convertir en intégrations vectorielles à l'aide d'un grand modèle de langage (modèle d'intégration).

Vous pouvez charger les données à partir de fichiers locaux ou de tables de base de données.

**Fichiers locaux**

Sélectionnez cette option si vous prévoyez de charger du texte, des documents, des données structurées ou tabulaires depuis votre ordinateur.

Ces fichiers peuvent inclure des fichiers texte bruts, des articles, des livres, des avis des utilisateurs, des commentaires des clients et de la documentation interne, qui sont souvent utilisés pour la recherche sémantique et les applications de génération augmentée par récupération (RAG) dans l'IA.

Vous pouvez glisser-déposer ou parcourir les fichiers depuis votre ordinateur local.

**Remarque** : vous pouvez charger 10 fichiers au maximum (limite de 100 Mo) avec le même schéma.

**Tables de base de données**

Sélectionnez cette option si vous souhaitez spécifier la table de base de données et les colonnes. En fonction de votre sélection (Segments uniquement ou Segments et intégrations), différentes sélections de tables, colonnes et intégrations s'affichent.
