Affichez les mesures et les informations sur le propriétaire d'un déploiement à l'aide de l'onglet Observabilité.

Sélectionnez le déploiement pour afficher les différents onglets.

**Tableau de bord** : les fiches récapitulatives des éléments suivants :

- Nombre total de traces (demandes traitées)

- Nombre de jetons (nombre total de jetons consommés)

- Latence P50 (temps de réponse moyen)

- Taux d'erreurs

**Traces** : affichez les versions disponibles, l'état, etc. Vous pouvez appliquer des filtres pour affiner la recherche de traces. Vous pouvez exécuter l'agent et afficher l'entrée/la sortie et les métadonnées correspondantes.

L'onglet Traces affiche les informations suivantes :

- Durée par demande.

- Jetons d'entrée/de sortie

- Détails des erreurs pour les demandes ayant échoué

- Étendues présentant les étapes de l'agent internes

**Étendue** : représente une opération individuelle dans une trace.

**Mesures** : représente les mesures quantitatives utilisées pour évaluer les performances du modèle.
