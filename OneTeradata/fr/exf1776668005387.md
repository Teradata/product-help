Une fois que vous avez sélectionné le déploiement requis, vous pouvez afficher ses détails dans différents onglets.

**Présentation** : affichez les détails du déploiement, tels que les détails de point de terminaison, le fichier JSON de configuration, etc.

**Référence de l'API** : affichez le code de l'API dans Python, Shell et LangChain.

Nous pouvons ajouter un utilisateur en tant que collaborateur sur un déploiement spécifique.

Cet utilisateur peut alors gérer le déploiement, de la même façon que la personne qui a créé le déploiement.

**Utilisateurs** : vous pouvez ajouter un utilisateur en tant que collaborateur sur un déploiement spécifique en cliquant sur le signe + et en fournissant l'ID d'utilisateur, puis en confirmant la création de l'utilisateur. L'utilisateur qui vient d'être ajouté peut alors gérer le déploiement, de la même façon que la personne qui a créé le déploiement.

Pour supprimer l'utilisateur, sélectionnez Supprimer et confirmez. Une fois les modifications appliquées, vous pouvez afficher la liste d'utilisateurs rectifiée pour le déploiement sélectionné.

**Journaux** : affichez les journaux du déploiement, y compris l'heure de création, les états des différents processus, etc. Si nécessaire, vous pouvez télécharger les journaux.
