AgentOps dans AI Studio est destiné à gérer, déployer et surveiller les agents d'IA. Il fournit un tableau de bord centralisé pour assurer le suivi des agents déployés, gérer les versions des agents et configurer les déploiements dans les environnements.

Les principaux cas d'utilisation d'AgentOps sont présentés ci-après.

Agents avec mémoire activée : les agents peuvent conserver l'historique des conversations et le contexte entre les sessions.

Intégration de base de données Teradata : créez des agents d'IA pouvant interroger et analyser les données dans les bases de données Teradata.

Automatisation des processus métier en plusieurs étapes à l'aide d'agents d'IA : agents pouvant collecter les informations et faire des recommandations.
