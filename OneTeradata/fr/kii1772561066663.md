Les modèles de langage (également appelés modèles de discussion) sont utilisés pour créer une réponse en langage naturel lors de la récupération. Lorsque vous posez une question en langage naturel aux données, le modèle de langage utilise les K meilleurs résultats renvoyés par l'algorithme de recherche efficace et fournit une réponse précise en langage naturel.

Vous pouvez sélectionner le modèle AWS ou le modèle Azure, ou choisir ultérieurement. Si vous sélectionnez le modèle AWS, vous pouvez choisir entre **Anthropic Claude 3 Haiku** et **Anthropic Claude 3.5 Sonnet**. Si vous sélectionnez le modèle Azure, vous pouvez sélectionner **text-embedding-ada-002**, **text-embedding-3-small** ou **text-embedding-3-large** pour l'intégration. Vous pouvez également sélectionner **gpt-4.1-mini** pour le modèle de langage (discussion).
