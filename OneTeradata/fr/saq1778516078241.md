Configurez l'environnement de test pour interagir avec les modèles, tester les invites et comparer le comportement des modèles dans un environnement de discussion contrôlé.

**Condition préalable :** une clé virtuelle est nécessaire. Si vous n'en avez pas, contactez votre administrateur.

1.  Dans le panneau de navigation de gauche, sélectionnez **Playground**.


1.  Dans **Source de la clé API**, saisissez votre clé virtuelle.


1.  Dans **Sélectionner un modèle**, sélectionnez un modèle associé à votre clé virtuelle.


1.  Vérifiez que le **Type de point de terminaison** est renseigné.


Votre environnement de test est prêt. Dans la section **Discussion**, entrez une invite afin d'évaluer comment le modèle répond. Pour ajuster les paramètres, reportez-vous à la section [Ajuster les paramètres de discussion](wdn1778518249826.md).
