En tant qu'analyste ou administrateur de données, vous pouvez gérer les collections de vecteurs que vous avez créées ou auxquelles vous avez accès.

Sur cette page, vous pouvez rechercher des collections actives en utilisant différents paramètres, tels que le nom de la collection, la description, le type de données, etc.

Vous pouvez également appliquer des filtres pour affiner les longues listes et trouver rapidement les éléments pertinents.

En fonction des privilèges de votre compte, vous pouvez :

Créer

Supprimer

Ajouter des données

Afficher les détails, puis les modifier

Effectuer une nouvelle tentative avec une collection de vecteurs ayant échoué, après avoir vérifié l'erreur correspondante

**États**

Voici les différents états :

**Prêt** : prêt à utiliser la collection de vecteurs.

**Création** : création en cours pour la collection de vecteurs.

**Échec** : la création de la collection de vecteurs a échoué avec une erreur. Affichez les détails de l'erreur et apportez les corrections nécessaires avant de réessayer.

Vous pouvez trier les collections de vecteurs en fonction de leur état.
