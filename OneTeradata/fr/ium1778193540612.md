Un noyau est un moteur ou un interpréteur qui exécute le code dans votre notebook. Chaque noyau dans les notebooks natifs Teradata prend en charge les langages suivants :

-   Python


-   Teradata SQL


Dans un noyau, vous pouvez effectuer les opérations suivantes :

-   Exécuter du code depuis votre notebook


-   Gérer les variables et effectuer le suivi de l'état des sessions


-   Générer une sortie après avoir exécuté votre code


Lorsque vous exécutez une cellule, le noyau interprète le code, l'exécute et renvoie les résultats à votre notebook.
