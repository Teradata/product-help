La page d'accueil de Teradata AI Studio Enterprise MCP fournit des indications rapides dans le produit pour vous aider à vous connecter au serveur Teradata Enterprise MCP et l'utiliser. Utilisez la page d'accueil pour obtenir des informations sur la connexion, générer des jetons d'authentification et accéder à des exemples d'intégration.

La page d'accueil fournit les informations suivantes. Pour savoir comment utiliser Teradata Enterprise MCP et les outils MCP, reportez-vous au [Guide de l'utilisateur de Teradata Enterprise MCP](https://docs.teradata.com/access?resourceid=enterprise-mcp-book).

-   **Point de terminaison du serveur**

    Ajoutez le point de terminaison du serveur MCP à votre notebook ou à la configuration d'outil externe pour vous connecter au serveur Enterprise MCP.

    -   **Serveur MCP par défaut**

        Pour vous connecter au serveur MCP par défaut, utilisez l'URL de serveur affichée dans l'UI.


    -   **Serveur MCP déployé par l'administrateur**

        Récupérez l'URL du point de terminaison du serveur à l'aide de l'API **Répertorier tous les serveurs**. Pour plus d'informations, consultez la section [Répertorier tous les serveurs](https://docs.teradata.com/access?resourceid=mcp-srvr-mg-list-servers&utm_source=console&utm_medium=iph).

    Pour accéder à ce point de terminaison, vous devez être connecté au réseau privé sur lequel AI Studio est déployé.


-   **Générer un jeton**

    Vous aurez besoin d'un jeton pour authentifier les demandes. Sélectionnez une méthode d'authentification. Pour plus d'informations, consultez la section [Autorisation, contrôle d'accès basé sur les rôles (RBAC) et authentification](https://docs.teradata.com/access?resourceid=author-rbac-authen).


-   **Connecter les clients externes**

    Connectez les clients MCP externes au serveur pour étendre l'accès MCP aux outils et ressources locaux. Pour plus d'informations, consultez la section [Connecter des clients MCP au serveur Enterprise MCP](https://docs.teradata.com/access?resourceid=mcp-connect-agent).
