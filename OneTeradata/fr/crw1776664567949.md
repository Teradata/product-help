Vous pouvez importer des agents à partir du référentiel GIT. Vous devez fournir un nom d'utilisateur et un mot de passe comme identifiants, les métadonnées des agents et tester la connexion. Une fois la connexion établie sans erreurs, importez les agents en sélectionnant Importer.

Un exemple de module et de sa structure est présenté ci-dessous.

agent-package

├── agent.py \# Required - Point d'entrée principale de l'agent

├── config.yaml \# Required - Configuration de l'agent

├── requirements.txt \# Required - Dépendances Python

├── modules/ \# Optional - Dépendances Python supplémentaires

│ └── *.py

└── resources/ \# Optional - Ressources supplémentaires

Si une erreur se produit, vérifiez les détails fournis et réessayez.
