La méthodologie d'opérations d'apprentissage automatique (MLOps) unifie le développement de modèles d'apprentissage automatique avec le déploiement et la gestion opérationnels. ModelOps est un sous-ensemble de MLOps qui se concentre sur le déploiement, la surveillance et la gestion des modèles d'apprentissage automatique dans les environnements de production. Il comble le fossé entre le développement des modèles et leur utilisation opérationnelle, garantissant ainsi une intégration fiable des modèles dans les processus et applications métier.

AI Studio ModelOps fournit des capacités opérationnelles pour les modèles d'apprentissage automatique personnalisés et d'analyse avancée, ainsi que pour les modèles d'IA fondamentaux tels que les grands modèles de langage hébergés, garantissant une gestion cohérente du cycle de vie pour tous les types de modèles pris en charge.

En outre, AI Studio ModelOps fournit une interface utilisateur Web facile à utiliser, une interface de ligne de commande et un kit de développement logiciel (SDK) Python ou R pour déployer et opérationnaliser des modèles analytiques avancés dans Teradata Vantage en utilisant le processus Analytics 1-2-3.

## Analytics 1-2-3


Analytics 1‑2‑3 est une approche qui vous aide à préparer les données, à entraîner les modèles et à déployer les modèles sur la plateforme Vantage. Le processus comprend trois étapes :

Préparation des données

Préparez les données d'entreprise à la modélisation à l'aide de la plateforme Vantage. Connectez-vous aux sources de données d'entreprise et appliquez des fonctions dans la base de données pour l'intégration, l'exploration, le nettoyage et l'ingénierie des caractéristiques.

Entraîner le modèle

Utilisez les outils, les langages et les bibliothèques que vous utilisez fréquemment. Vous pouvez entraîner des modèles à l'aide d'infrastructures telles que SAS, scikit‑learn, XGBoost, MLlib ou H2O.

Déployer le modèle

ModelOps permet d'intégrer votre propre modèle (BYOM) entraîné en externe dans Vantage ou de suivre le processus d'entraînement. Le modèle déployé s'exécute dans Vantage pour générer des prédictions et est surveillé tout au long de son cycle de vie.
