Un pool de nœuds désigne un groupe de serveurs partagé. Chaque pool a des spécifications différentes. Sélectionnez le pool qui correspond à votre charge de travail.

1.  Sélectionnez un pool de nœuds.

    1.  Calcul général : pour les requêtes SQL, l'analyse exploratoire dans Python et les fonctions dans la base de données.

        5 nœuds de 16 vCPU, mémoire : 64 Go de RAM


    1.  Mémoire élevée : lorsque votre charge de travail charge entièrement dans la mémoire de grands ensembles de données : DataFrames de plusieurs giga-octets ou matrices de fonctionnalités qui épuisent le calcul général.

        3 nœuds de 64 vCPU, mémoire : 512 Go de RAM


1.  Configurez les ressources pour les nœuds réguliers ou GPU.

    1.  [Configurer les ressources pour les nœuds réguliers](vzi1778132810791.md)


    1.  [Configurer les ressources pour les nœuds GPU](axj1778133448257.md)
