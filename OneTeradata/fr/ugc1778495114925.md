La plate-forme de modèles d'IA vous permet d'accéder aux modèles de référence approuvés pour être utilisés dans votre organisation, de les gérer et de les surveiller.

Pour analyser les modèles de référence approuvés pour votre organisation, utilisez l'onglet **Présentation**. Comparez les capacités, vérifiez les limites de jetons et identifiez les modèles adaptés pour votre environnement.

**Cartes de modèle**

Consultez un instantané général de chaque modèle pour comparer rapidement les capacités.

**Limites de jetons**

Les informations sur chaque carte vous permettent de comprendre combien d'entrées un modèle accepte et combien de sorties il génère.

-   **Nombre maximal de jetons d'entrée**

    Texte maximal que vous pouvez envoyer dans une demande. Ce paramètre définit la fenêtre contextuelle du modèle.


-   **Nombre maximal de jetons de sortie**

    Texte maximal que le modèle peut générer dans une réponse.


## Qui fait quoi


-   **Administrateur**

    Vous pouvez ajouter un modèle à partir de cette page. Reportez-vous à la section [Ajouter un modèle](fdy1778502087907.md).


-   **Utilisateur**

    Saisissez votre clé virtuelle pour afficher les modèles qui vous sont attribués.
