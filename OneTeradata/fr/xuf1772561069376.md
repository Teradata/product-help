Vous pouvez ajouter une nouvelle source de données ou supprimer une source de données existante et enregistrer la collection de vecteurs.

Utilisez les filtres pour rechercher une source de données spécifique et la modifier ou la supprimer selon vos besoins.

**Remarque** : vous ne pouvez pas supprimer toutes les sources de données de l'onglet. Au moins une source de données doit être disponible.
