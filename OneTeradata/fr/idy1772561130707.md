Définissez la collection de vecteurs avec un nom unique et fournissez une description appropriée.

Pour créer une collection de vecteurs, vous devez sélectionner comme type de données des fichiers PDF de votre stockage local ou des informations présegmentées à partir d'une table de base de données, ou des intégrations préexistantes (données vectorielles).

Une fois terminé, sélectionnez l'emplacement de la base de données pour stocker la nouvelle collection. Par défaut, la même base de données que celle avec laquelle vous vous êtes connecté est utilisée.
