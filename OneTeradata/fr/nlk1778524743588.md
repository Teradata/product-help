La configuration complète des clés virtuelles s'affiche dans le format JSON en lecture seule. Cela inclut les limites, les autorisations, les métadonnées, les politiques et les valeurs par défaut appliquées par le système. Utilisez cette vue pour déboguer les problèmes, permettre les audits ou référencer les détails des clés dans les flux de travail d'automatisation.
