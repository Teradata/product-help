Vous pouvez sélectionner les algorithmes de recherche et également définir les paramètres de recherche à partir de cette page.

En fonction de vos sélections, différentes options s'offrent à vous.

## Vecteur-Distance


Lorsque vous sélectionnez Vecteur-Distance, dans les paramètres de recherche, vous pouvez conserver les valeurs par défaut ou sélectionner les valeurs requises en désactivant l'option Utiliser les valeurs par défaut. Vous pouvez modifier les éléments suivants :

**Correspondances de similarité** : sélectionnez le nombre (entre 1 et 1 024) de correspondances de similarité à générer.

**Seuil de similarité **: permet de prendre en compte les tables et vues correspondantes.

**Paramètres généraux** : sélectionnez Euclidien, Cosinus ou Produit scalaire.

• **Euclidien** : calcule la similarité en utilisant la distance euclidienne entre les vecteurs.

• **Cosinus** : calcule la similarité cosinus (produit scalaire normalisé) entre deux vecteurs.

• **Produit scalaire** : calcule la similarité entre vecteurs à l'aide du produit scalaire.

## K-moyennes


Lorsque vous sélectionnez K-moyennes, vous pouvez conserver les valeurs par défaut ou sélectionner les valeurs requises en désactivant l'option **Utiliser les valeurs par défaut**. Vous pouvez modifier les éléments suivants :

**Paramètres d'indexation** : sélectionnez les valeurs requises, telles que le nombre de clusters à entraîner, le nombre d'itérations à exécuter, la valeur initiale, l'algorithme d'initialisation des centroïdes (aléatoire ou K-moyennes++), etc.

**Paramètres de recherche** : sélectionnez le cluster d'entraînement à compter, les correspondances de similarité et les valeurs de seuil.

**Paramètres généraux** : sélectionnez Euclidien, Cosinus ou Produit scalaire

## HNSW


Lorsque vous sélectionnez HNSW, vous pouvez conserver les valeurs par défaut ou sélectionner les valeurs requises en désactivant l'option **Utiliser les valeurs par défaut**. Vous pouvez modifier les éléments suivants :

**Paramètres d'indexation** : sélectionnez le nombre de couches du graphique, le nombre de connexions de nœuds, les valeurs de construction EF, etc.

**Paramètres de recherche** : sélectionnez les correspondances de similarité et les valeurs de recherche EF.

**Paramètres généraux** : sélectionnez Euclidien, Cosinus ou Produit scalaire.
