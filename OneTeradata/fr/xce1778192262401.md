|Type de cellule|Objet|Comportement|
|----------|--------|---------|

|Cellule Markdown|Ajouter de la documentation et du texte formaté|Restitue la syntaxe Markdown dans un texte stylisé|
|SQL|Écrire et exécuter le code SQL Teradata|Le noyau exécute le code SQL et affiche la sortie|
