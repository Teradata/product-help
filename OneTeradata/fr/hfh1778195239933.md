Comparaison entre les notebooks Jupyter et les notebooks natifs Teradata.

|Catégorie|Notebooks natifs Teradata|Notebooks Jupyter|
|---------|--------------------------|------------------|
|Noyau SQL|Noyau SQL natif|Inclus|
|Navigation dans la base de données|Navigateur + gestionnaire de connexions|Aucun|
|Analyses dans la base de données|Teradataml / tdplyr intégré|Nécessite une configuration manuelle|
|Visualisation|Vega + widgets d'IU (aucune bibliothèque Python requise)|Matplotlib/seaborn/plotly|
|IU ML/BYOM|teradatamlwidgets (AutoML, magasin de caractéristiques, magasin de vecteurs, BYOM)|Aucun|
|Effort de configuration|Faible|Élevé|
|Idéal pour|SQL Teradata + Analyses Vantage + flux de travail d'entreprise|Science des données générale|
