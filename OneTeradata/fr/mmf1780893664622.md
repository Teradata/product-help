L'explorateur de données vous permet de parcourir vos bases de données Teradata et leurs objets directement depuis la barre latérale du notebook.

Vous pouvez effectuer les opérations suivantes :

• Développer une base de données pour afficher les objets organisés par type

• Développer une table ou une vue pour afficher ses colonnes ainsi que ses types de données

• Rechercher : filtrer la base de données en temps réel pour trouver des objets par leur nom

• Marquer d'une étoile/Annuler le marquage d'une étoile : marquer d'une étoile une base de données afin de l'épingler en haut du panneau pour y accéder rapidement

• Actualiser : recharger le dernier état d'une base de données, d'une table ou d'une vue
