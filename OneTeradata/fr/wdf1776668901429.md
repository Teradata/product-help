Pour déployer un agent, sélectionnez-le dans le registre d'agents.

La procédure est la suivante :

**Détails de l'agent** : sélectionnez parmi les agents disponibles pour être déployés. Une fois l'agent sélectionné, toutes les versions disponibles sont affichées. Sélectionnez la version requise pour afficher les détails.

Sélectionnez **Suivant**.

**Métadonnées** : sélectionnez les métadonnées et fournissez le nom et la description.

**Configurer les points de terminaison** : affichez le fichier JSON respectif. Vous pouvez le modifier si cela est nécessaire.

**Ressources** : affichez les paramètres des ressources et configurez les ressources. Dans Configurer les ressources, vous pouvez choisir Standard ou Personnalisé.

**Résumé** : affichez le résumé de l'agent que vous souhaitez déployer.

**Déployer** : pour déployer l'agent. Vous pouvez afficher la progression, l'état et d'autres informations.

Remarque : lorsque vous déployez des agents, il n'est pas nécessaire de les importer. Vous importez un agent uniquement lorsque vous l'ajoutez au registre et créez d'autres versions.
