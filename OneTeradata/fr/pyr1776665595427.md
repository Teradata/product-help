Vous pouvez importer des agents depuis l'option Fichiers locaux en sélectionnant le fichier ZIP requis, puis en le chargeant. Assurez-vous de fournir les métadonnées de l'agent avant de continuer.

**Remarque :** seuls les fichiers ZIP ou dont la taille ne dépasse pas 100 Mo sont autorisés.

Un exemple de module et de sa structure est présenté ci-dessous.

agent-package.zip

├── agent.py \# Required - Point d'entrée principale de l'agent

├── config.yaml \# Required - Configuration de l'agent

├── requirements.txt \# Required - Dépendances Python

├── modules/ \# Optional - Dépendances Python supplémentaires

│ └── *.py

└── resources/ \# Optional - Ressources supplémentaires

Si une erreur se produit, vérifiez les détails fournis et réessayez.

Les erreurs les plus courantes peuvent être un serveur indisponible, des identifiants incorrects ou des problèmes d'accès.
