Effectuez des recherches dans les modèles d'IA enregistrés dans la plate-forme de modèles d'IA, analysez-les et comparez-les. Utilisez les détails des modèles pour évaluer les capacités et identifier les options qui correspondent à vos flux de travail.

## Ce que la page affiche


-   **Nom**

    Affiche le nom du modèle.


-   **ID ou alias de modèle**

    Affiche l'identifiant unique ou l'alias attribué au profil du modèle.


-   **Version**

    Affiche le numéro de version du modèle, le cas échéant.


-   **Type**

    Affiche la catégorie du modèle. Par exemple, intégration pour les modèles d'intégration de texte ou d'image, ou discussion pour les modèles conversationnels.


-   **Fournisseur**

    Affiche le fournisseur d'infrastructure qui héberge le modèle. Par exemple, Bedrock.


-   **Description**

    Affiche une brève description du modèle.


-   **URL de base de l'API**

    Affiche l'URL de base du point de terminaison de l'API du modèle, s'il est configuré.


-   **Longueur du contexte**

    Affiche le nombre maximal de jetons que le modèle peut traiter dans une demande unique, s'il est spécifié.


-   **Limite de débit**

    Affiche le nombre maximal de demandes autorisé dans une période définie, s'il est spécifié.


-   **Mis à jour à**

    Affiche la date et l'heure de la dernière mise à jour de l'entrée du modèle.


## Qui fait quoi


-   **Administrateur**

    Afficher tous les modèles enregistrés.


-   **Utilisateur**

    Afficher uniquement les modèles attribués à votre clé virtuelle. Pour demander l'accès à des modèles supplémentaires, contactez votre administrateur.
