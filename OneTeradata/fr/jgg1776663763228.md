Dans AI Studio, ouvrez AgentOps pour afficher un tableau de bord centralisé pour assurer le suivi des agents déployés, gérer les versions des agents et configurer les déploiements dans les environnements.

Cette page vous permet d'effectuer les opérations suivantes :

• Vérifier les déploiements existants et les entrées de registre de l'agent.

• Enregistrer un nouvel agent.

• Tester l'agent à l'aide de l'environnement de test.

• Afficher les mesures des agents dans l'onglet Observabilité.
