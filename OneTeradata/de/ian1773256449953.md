Die Startseite von Teradata AI Studio Enterprise MCP bietet schnelle Anleitung direkt in der Anwendung, damit Sie einfacher eine Verbindung zum Teradata Enterprise MCP-Server aufbauen und ihn verwenden können. Verwenden Sie die Startseite, um Verbindungsinformationen zu erhalten, Authentifizierungstokens zu generieren und auf Integrationsbeispiele zuzugreifen.

Die Startseite bietet die folgenden Informationen. Weitere Informationen zur Arbeit mit Teradata Enterprise MCP und MCP-Tools finden Sie im [Benutzerhandbuch für Teradata Enterprise MCP](https://docs.teradata.com/access?resourceid=enterprise-mcp-book).

-   **Serverendpunkt**

    Fügen Sie den MCP-Serverendpunkt zu einer Konfiguration eines Notebooks oder eines externen Tools hinzu, um eine Verbindung mit dem Enterprise MCP-Server herzustellen.

    -   **Standardmäßiger MCP-Server**

        Verwenden Sie die auf der Benutzeroberfläche angezeigte Server-URL, um eine Verbindung mit dem standardmäßigen MCP-Server herzustellen.


    -   **Vom Administrator bereitgestellter MCP-Server**

        Rufen Sie die URL des Serverendpunkts mithilfe der API **Alle Server auflisten** ab. Weitere Informationen finden Sie unter [Alle Server auflisten](https://docs.teradata.com/access?resourceid=mcp-srvr-mg-list-servers&utm_source=console&utm_medium=iph).

    Sie müssen mit dem privaten Netzwerk verbunden sein, in dem AI Studio bereitgestellt ist, um auf diesen Endpunkt zuzugreifen.


-   **Ein Token generieren**

    Sie benötigen ein Token, um Anfragen zu authentifizieren. Wählen Sie eine Authentifizierungsmethode aus. Weitere Informationen finden Sie unter [Autorisierung, RBAC und Authentifizierung](https://docs.teradata.com/access?resourceid=author-rbac-authen).


-   **Externe Clients verbinden**

    Verbinden Sie externe MCP-Clients mit dem Server, um den MCP-Zugriff auf lokale Tools und Ressourcen auszuweiten. Weitere Informationen finden Sie unter [MCP-Clients mit dem Enterprise MCP-Server verbinden](https://docs.teradata.com/access?resourceid=mcp-connect-agent).
