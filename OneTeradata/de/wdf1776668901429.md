Wählen Sie zum Bereitstellen eines Agenten den bereitzustellenden Agenten in der Agentenregistrierung aus.

Befolgen Sie die folgenden Schritte:

**Agentendetails**: Wählen Sie für die Bereitstellung aus den verfügbaren Agenten aus. Wenn Sie eine Auswahl getroffen haben, werden alle verfügbaren Versionen angezeigt. Wählen Sie die erforderliche aus, um die Details anzuzeigen.

Wählen Sie **Weiter** aus.

**Metadaten**: Wählen Sie die Metadaten aus und geben Sie Namen und Beschreibung an.

**Endpunkte konfigurieren**: Zeigt die entsprechende JSON-Datei an. Sie können sie bei Bedarf bearbeiten.

**Ressourcen**: Zeigen Sie die Ressourceneinstellungen an und konfigurieren Sie die Ressourcen. In „Ressourcen konfigurieren“ können Sie entweder „Standard“ oder „Benutzerdefiniert“ auswählen.

**Zusammenfassung**: Zeigt die Zusammenfassung des Agenten an, den Sie bereitstellen möchten.

Nutzen Sie **Bereitstellen**, um den Agenten bereitzustellen. Sie können Fortschritt, Status und andere Details anzeigen.

Hinweis: Sie müssen keinen Import durchführen, wenn Sie Agenten bereitstellen. Ein Import ist nur erforderlich, wenn Sie der Registrierung einen Agenten hinzufügen und wenn Sie mehrere Versionen erstellen.
