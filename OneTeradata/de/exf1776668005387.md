Wenn Sie die erforderliche Bereitstellung ausgewählt haben, können Sie die Details dazu in verschiedenen Registerkarten einsehen.

**Übersicht**: Details zur Bereitstellung anzeigen, etwa Endpunktdetails, Konfigurations-JSON und mehr.

**API-Referenz**: API-Code in Python, Shell und LangChain anzeigen.

Wir können einen Benutzer als Mitarbeitenden für eine bestimmte Bereitstellung hinzufügen.

Dieser Benutzer kann die Bereitstellung dann verwalten, ähnlich wie die Person, die die Bereitstellung erstellt hat

**Benutzer**: Wenn Sie einen Benutzer als Mitarbeitenden für eine bestimmte Bereitstellung hinzufügen möchten, verwenden Sie „+“, geben Sie die Benutzer-ID an und bestätigen Sie den Benutzer. Der neu hinzugefügte Benutzer kann die Bereitstellung nun verwalten, ähnlich wie der Benutzer, der die Bereitstellung erstellt hat.

Sie können den Benutzer auch entfernen, indem Sie „Löschen“ auswählen und bestätigen. Wenn die Änderungen übernommen werden, können Sie die aktualisierte Benutzerliste für die ausgewählte Bereitstellung anzeigen.

**Protokolle**: Protokolle der Bereitstellung anzeigen, einschließlich Erstellungszeit, Status verschiedener Prozesse usw. Bei Bedarf können Sie die Protokolle herunterladen.
