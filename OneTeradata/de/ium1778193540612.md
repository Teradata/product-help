Ein Kernel ist eine Engine oder ein Interpretierprogramm, das den Code auf Ihrem Notebook ausführt. Jeder Kernel auf nativen Teradata-Notebooks unterstützt die folgenden Sprachen:

-   Python


-   Teradata SQL


Innerhalb eines Kernels haben Sie folgende Möglichkeiten:

-   Code auf Ihrem Notebook ausführen


-   Variablen verwalten und den Sitzungsstatus überwachen


-   Ausgaben nach der Ausführung Ihres Codes generieren


Wenn Sie eine Zelle ausführen, interpretiert der Kernel den Code, führt ihn aus und sendet das Ergebnis zurück an Ihr Notebook.
