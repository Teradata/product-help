Eine Zelle ist ein Behälter für Code oder Text innerhalb Ihres Notebooks. Sie enthält ausführbaren Code (Python, Teradata SQL) und Markdown-Text für Dokumentation oder Erklärungen.

|Zelltyp|Zweck|Verhalten|
|----------|--------|---------|
|Codezelle|Python-Code schreiben und ausführen|Kernel führt den Python-Code aus und zeigt die Ausgabe an|
|Markdown-Zelle|Dokumentation und formatierten Text hinzufügen|Stellt Markdown-Syntax als stilisierten Text dar|
|SQL|Teradata SQL-Code schreiben und ausführen|Kernel führt den SQL-Code aus und zeigt die Ausgabe an|
