Auf dieser Seite können Sie die Suchalgorithmen auswählen und Suchoptionen festlegen.

Je nach Auswahl stehen Ihnen verschiedene Optionen zur Verfügung.

## FLAT


Wenn Sie „FLAT“ auswählen, können Sie in den Sucheinstellungen die Standardeinstellungen beibehalten oder die gewünschten Werte auswählen, indem Sie die Option „Standardeinstellungen verwenden“ deaktivieren. Sie können folgende Einstellungen ändern:

**Ähnlichkeitstreffer**: Wählen Sie die Anzahl (zwischen 1 und 1024) der zu generierenden Ähnlichkeitstreffer aus. Verwenden Sie einen niedrigen Wert, wenn Sie möglichst exakte Treffer benötigen, oder um die Leistung zu optimieren und gleichzeitig die Kosten zu senken.

**Ähnlichkeitsschwelle**: Zur Berücksichtigung übereinstimmender Tabellen und Ansichten. Diese Option dient als Qualitätsfilter und gibt nur relevante Ergebnisse oberhalb einer bestimmten Ähnlichkeitsbewertung oder unterhalb eines Entfernungswerts aus anstatt einfach die obersten \\(k\\) Treffer.

**Allgemeine Einstellungen**: Wählen Sie „Euklidisch“, „Kosinus“ oder „Skalarprodukt“ aus.

• **Euklidisch**: Berechnet die Ähnlichkeit anhand des euklidischen Abstands zwischen den Vektoren.

• **Kosinus**: Berechnet die Kosinusähnlichkeit (normiertes Skalarprodukt) zwischen zwei Vektoren. Verwenden Sie diese Option für die semantische Suche in eingebetteten Texten, den Dokumentvergleich und Empfehlungssysteme, bei denen die Größenordnung keine Rolle spielt.

• **Skalarprodukt**: Berechnet die Ähnlichkeit zwischen Vektoren mithilfe des Skalarprodukts. Nutzen Sie diese Option, wenn Beliebtheit (Größenordnung) eine größere Rolle spielen soll, oder für normalisierte Vektorsuche.

## IVF_FLAT


Bei Auswahl von „IVF_FLAT“ können Sie die Standardeinstellungen beibehalten oder die gewünschten Werte auswählen, indem Sie die Option **Standardeinstellungen verwenden** deaktivieren. Sie können folgende Einstellungen ändern:

**Indexierungseinstellungen**: Wählen Sie die gewünschten Werte aus, wie z. B. die Anzahl der zu trainierenden Cluster, die Anzahl der durchzuführenden Iterationen, den Startwert für die Clusterzentren sowie den Algorithmus für Schwerpunktinitialisierung (Random oder Kmeans++) usw. Dient hauptsächlich dazu, die Ähnlichkeitssuche durch Unterteilung von Daten in Cluster zu beschleunigen; so wird eine ANN-Suche (Approximate Nearest Neighbor) anstelle einer exakten, langsamen Suche erstellt.

**Sucheinstellungen**: Wählen Sie den zu berücksichtigenden Trainings-Cluster, die Ähnlichkeitskriterien und die Schwellenwerte aus.

**Allgemeine Einstellungen**: Wählen Sie „Euklidisch“, „Kosinus“ oder „Skalarprodukt“ aus.

## HNSW


Bei Auswahl von „HNSW“ können Sie die Standardeinstellungen beibehalten oder die gewünschten Werte auswählen, indem Sie die Option **Standardeinstellungen verwenden** deaktivieren. Sie können folgende Einstellungen ändern:

**Indexierungseinstellungen**: Wählen Sie die Anzahl der Ebenen für die Grafik, die Anzahl der Knotenverbindungen, die EF-Konstruktionswerte usw. aus.

**Sucheinstellungen**: Wählen Sie die zu verwendenden Ähnlichkeitstreffer und die EF-Suchwerte aus.

**Allgemeine Einstellungen**: Wählen Sie „Euklidisch“, „Kosinus“ oder „Skalarprodukt“ aus.
