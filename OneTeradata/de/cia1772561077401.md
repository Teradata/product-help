Wählen Sie das gewünschte Einbettungsmodell aus, das Sie verwenden möchten. Verfügbare AWS-Modelle:

· Amazon Titan Text Embeddings V2-Modell

· Amazon Titan Text Embeddings V1-Modell

· Amazon Titan Image Embeddings V1-Modell

**Hinweis**: Bei einer Änderung des Einbettungsmodells werden alle vorhandenen Daten in der Vektorsammlung neu eingebettet, und gleichzeitig werden alle neuen Daten eingebettet.

Sie können auch die verfügbaren Azure-Modelle nutzen.

· Tex Embedding Ada 002

· Tex Embedding 3 Small

· Tex Embedding 3 Large

Außerdem können Sie als Modell für Sprache (Chat) **gpt-4.1-mini** auswählen.
