### Managing Users

#### Types of Users

| Service Portal |Access |Console|Access  |
|--|--|--|

| Security Manager |Read-Only |Site Access Manager  |Full |
| Access Manager |Read-Only |Site Access Manager  |Full |
| Technical Contact |Full|Site Admin |Full |
